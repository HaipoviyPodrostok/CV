\end{itemize}

\href{https://github.com/HaipoviyPodrostok/Prompter-glasses} {\textbf{\color{primary} Прототип очков дополненной реальности}} \faGithub
\hfill \textcolor{secondary}{С++, Fusion 360}
\begin{itemize}
\item Рабочий функциональный прототип очков дополненной реальности. В данной реализации используются как очки-телесуфлеры. 
\item Очки-суфлёры — устройство, которое позволяет читать заготовленный текст. Текст проецируется на полупрозрачное зеркало и становится доступным для прочтения только для пользователя. 
\item Это решает проблему мобильности студийных телесуфлеров.

\end{itemize}

\href{https://github.com/HaipoviyPodrostok/LIDAR} {\textbf{\color{primary} Разработка алгоритма для обработки данных с LIDAR}} \faGithub
\hfill \textcolor{secondary}{C++}
\begin{itemize}
\item Обработка данных промышленного LiDAR-сканера для определения высоты загрузки кузова грузовика.
\item Разработка алгоритма для аппроксимации поверхности кузова грузовика плоскостью, устойчивого к тяжелым погодным условиям
\item Программа аппроксимирует плоскость дна кузова алгоритмом RANSAC с МНК-уточнением и генерирует карту отклонений (heatmap), визуализирующую инлайеры (дно) и выбросы (снег, пыль, груз).
\end{itemize}
